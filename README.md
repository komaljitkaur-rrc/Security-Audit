## IMPORTANCE OF PREDEFINED STANDARDS IN SOFTWARE DEVELOPMENT

**Software** **Development** **Standard**: It is set of best practices which is used to ensure that code is consistent and readable across development team.

**Predefined** **Standard** **of** **software** **development** such as **PEP** **8** or Python Enhanced Proposal is a style guide for python code that provide suggestions for coding conventions, such as naming convention, code formatting and documentation. Moreover,by following **PEP** **8** guidelines,developer can write a code which consistent, readable and easy to maintain.

## METHOD BY WHICH SOFTWARE DEVELOPER CAN EMPLOY ETHICAL HABITS WHEN WRITING CODE:

- By ensuring that their work is understandable and maintainable by others as well, for instance, in future if somebody want to use this code or want to modify it, that time they can easily do it.
- Getting reviews from other co-workers or team-mates for accountability and quality improvement.
- By using predifined standards of software development.
